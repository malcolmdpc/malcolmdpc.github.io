# Patrones Lab · Data & Analytics Portfolio

Repositorio de la web pública de **Patrones Lab**, un portfolio de proyectos de analítica de datos, Business Intelligence, Machine Learning, Python y dashboards.

La web reúne proyectos orientados a explorar datos, detectar patrones, construir visualizaciones y convertir información compleja en análisis claros y visuales.
